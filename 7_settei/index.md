# 管理画面にログインする

1. まずWordPressに管理者でログインする。
http://example.com/wp-admin/

![管理画面]({{ site.baseurl }}/assets/images/7-1-001.png)

2. 左ペインの「設定」から「一般設定」をクリックする。
