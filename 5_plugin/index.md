# プラグイン
プラグインとは、WordPressの機能を拡張する為のツールです。
この項目では、プラグインの有効化と新規追加方法についてご説明いたします。

## 有効化
(1)管理画面の「プラグイン」より「インストール済プラグイン」を選択します。
![管理画面の「プラグイン」より「インストール済プラグイン」を選択します。]({{ site.baseurl }}/assets/images/plugin01.png)
